Some Helpful Tools

Excel to NGDS FeatureClass

This is a tool for ArcMap which validates and converts a spreadsheet in an Excel file to a valid NGDS FeatureClass. It has primarily been tested in ArcGIS 10.0 but does run in 10.1.

Download the Package View the Source Code Read the Docs

Excel to LAS Converter

This is an executable python script that can convert data from a template Excel file into a LAS version 2.0 formatted XML file. An example template file can be found in the package after download.
