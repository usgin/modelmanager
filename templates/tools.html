Some Helpful Tools

Excel to NGDS Service ArcGIS Tool

This is a tool for ArcMap 10, 10.1 and 10.2 which validates and converts a spreadsheet in an Excel content model to a geodatabase ready to be deployed as a NGDS Service.

Download the Package View the Source Code Read the Docs

Well Log Data Tools

This is an executable python script which offers two different conversions for well log data: 1) Conversion from a BoreholeLASLogData.xls model to a LAS version 2.0 formatted XML file. 2) Conversion from a LAS 2.0 formatted XML file to an entry in the WellLog Content Model. Example templates for BoreholeLASLogData.xls and WellLogsTemplate.xls can be found in the package after download.

Download the Package View the Source Code Read the Docs

Auto Publisher

This is an executable program for ArcGIS 10.1 and 10.2 that publishes services from finished ArcMap projects. It can publish a single file or folder of files. Service properties are filled according to the user's saved preferences. The full list of capabilities can be found in the ReadMe.
